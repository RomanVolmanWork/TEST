Eclipse bundle readme.

Overview
========
This package contains the moviEclipse bundle, an C/C++ IDE for Myriad development.
The product is based on Eclipse Neon [1] and includes also an Oracle JRE [2]

Supported operating systems
===========================
To function properly you will need a system with at least the following characteristics:

CPU: 3.00 Ghz dual core processor
RAM: 4GB free memory
DISK: 1GB free disk space
OS: Windows 7 64 bit 
	or 
	Linux 64 bit with
		GTK+ 3.0.0 and its dependencies (for GTK+ 3) (see [3])

Please note that the tests has been performed only on Ubuntu but any Linux distribution with the above characteristics should work. 

Installation
===========
1. Download the "MoviEclipse-linux.gtk.x86_64-#.#.#.tar.gz" for Linux or "MoviEclipse-win32.win32.x86_64-#.#.#.zip" for Windows
2. Unpack the downloaded archive in a folder of your choice.
3. Execute "eclipse" for Linux or "eclipse.exe" for Windows.

Getting started
===============
If you use moviEclipse for the first time we recommend you to see the provided Myriad Cheetsheets:
- start Eclipse
- execute menu "Help/Cheet Sheats..."
- Select and run the cheat sheets under the "Myriad Development" section.
